# Contributing Guidelines
- fork
- create a new branch with `dev-${GITHUB-USERNAME}` name (e.g. `dev-AaroneGeorge`)
  - ```bash
    git checkout -b dev-AaroneGeorge
    ```
- commit your changes
  - ```bash
    ./pushme.sh -m "your commit msg"
    ```
- create a pull request
